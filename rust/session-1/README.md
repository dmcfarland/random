# Session one - Experimenting with Rust

## Introduction

Make sure rust is installed - https://doc.rust-lang.org/book/ch01-01-installation.html
Install brewfile - `brew bundle install`

Add .env file with
```

MONGO_DB=
MONGO_COLLECTION=
```

Review Makefile for commands to run the code.
